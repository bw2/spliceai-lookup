This tool computes SpliceAI scores by running the SpliceAI model on your variant(s).

Features:
- compute scores for any variant, including in-dels that aren't in precomputed tables published by the SpliceAI team
- see separate probabilities for each category - splice donor, splice acceptor - and not just for the category with the highest probability.
- adjust model parameters such as -D
